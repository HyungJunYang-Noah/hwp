# Document Family Model

Use this model when the user provides a new example document and wants future drafts to resemble it.

## Required Metadata

- `template_family`
- `review_purpose`
- `decision_style`

## Useful Metadata

- `audience`
- `evidence_pattern`
- `required_sections`
- `preferred_tone`
- `source_examples`

## Action Rule

- First example in a family:
  Record it in ARK and keep the adaptation local.
- Second example in the same family:
  Start a dedicated reference note or domain note.
- Third example in the same family:
  Create or tighten a preset and a validation rulebook.
