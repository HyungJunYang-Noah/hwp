# Workspace Samples

Use these bundled files when the user asks to match an existing 양식 or style.

## Primary Templates

- `assets/base_report_template.hwpx`: default generator-compatible review and report template.
- `assets/samples/blank_review_template.hwpx`: blanker variant for minimal-form drafting.
- `assets/reference_sample.hwpx`: compact packaged reference sample.

## Representative Examples

- `assets/samples/sample_minimal.hwpx`: small sample for quick structure checks.
- `assets/samples/completed_review_example.hwpx`: completed review example for heading rhythm, section density, and tone.
- A task-provided legacy `.hwp` file: use it as a structural reference when the user wants to follow an older Hancom family that is not covered by the bundled `.hwpx` samples.

## Selection Guidance

- Start with `assets/base_report_template.hwpx` for general 검토서, 보고서, and review-style drafts.
- Start with `assets/samples/blank_review_template.hwpx` when the user wants a sparse empty form.
- Compare against `assets/samples/completed_review_example.hwpx` when calibrating tone, section order, or heading phrasing.
- Extract headings and section order from a task-provided legacy `.hwp` sample, then produce a new `.hwpx` output instead of trying to patch the `.hwp` package directly.
